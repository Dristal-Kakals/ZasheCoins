# ЗащеКоины 🎰

Виртуальное казино на Go с графическим интерфейсом ([Ebiten](https://ebitengine.org/)).
Крути рулетку и открывай кейсы на виртуальные **защекоины** — настоящие деньги тут ни при чём.

## Мини-игры

- **Рулетка** — европейская рулетка (0–36). Ставь на красное/чёрное (x2) или на зеро (x14)
  и крути вращающееся колесо.
- **Слоты** — классический трёхбарабанный автомат. Три символа в ряд на центральной линии —
  большой множитель (от x5 за вишню до x50 за семёрку), любая пара — x2.
- **Кейсы** — три кейса (Деревянный / Стальной / Алмазный) с предметами разной редкости.
  Прокрутка ленты в стиле CS:GO, выпавший предмет зачисляется на баланс.

Баланс автоматически сохраняется в `zashecoins_save.json`. Старт — 1000 ©.
Если всё проиграл — в главном меню есть кнопка «Сбросить баланс».

## Запуск

```sh
go run ./src
```

## Сборка

```sh
go build -o zashecoins.exe ./src
```

> Требуется Go 1.21+ и C-компилятор (CGO) для графики Ebiten.

## Структура

| Файл | Назначение |
|------|------------|
| `src/main.go` | игровой цикл, сцены, верхняя панель с балансом |
| `src/state.go` | состояние игрока и сохранение на диск |
| `src/ui.go` | шрифты, цвета, кнопки, примитивы рисования |
| `src/controls.go` | контрол выбора ставки (фишки) |
| `src/scene_menu.go` | главное меню |
| `src/scene_roulette.go` | рулетка |
| `src/scene_slots.go` | слоты (барабаны + векторные символы) |
| `src/scene_cases.go` | кейсы |
